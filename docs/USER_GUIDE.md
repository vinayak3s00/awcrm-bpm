## 🚀 Getting Started

### First Time Setup

1. **Access the Setup Page**
   - Navigate to `/setup` when first accessing the system
   - Complete the organization setup form
   - Create your admin account

2. **Organization Configuration**
   - Enter your organization name
   - Provide admin contact details
   - Choose whether to create sample data

3. **Sample Data (Recommended)**
   - Select "Create sample data" for a quick start
   - Includes sample contacts, companies, deals, and activities
   - Helps you understand the system features

### Navigation Basics

- **Sidebar Navigation** - Main menu on the left
- **Global Search** - Press `Cmd/Ctrl + K` to search anywhere
- **Breadcrumbs** - Track your current location
- **Quick Actions** - Access common actions from any page

## 👥 Contact Management

### Creating Contacts

1. **Navigate to Contacts**
   - Click "Contacts" in the sidebar
   - Click the "+" button or "Add Contact"

2. **Fill Contact Information**
   - **Required**: First Name, Last Name
   - **Optional**: Email, Phone, Company, Job Title
   - **Status**: Prospect, Active, Inactive
   - **Source**: How you found this contact

3. **Additional Details**
   - Notes for important information
   - Tags for categorization
   - Custom fields for specific needs

### Managing Contacts

#### Viewing Contacts
- **List View** - Table with sortable columns
- **Search** - Use the search bar or global search
- **Filters** - Filter by status, source, or tags
- **Pagination** - Navigate through large contact lists

#### Editing Contacts
- Click on any contact to view details
- Use the "Edit" button to modify information
- Changes are saved automatically

#### Bulk Operations
- Select multiple contacts using checkboxes
- Available actions:
  - Bulk delete
  - Bulk status update
  - Bulk tag assignment

### Contact Details Page

#### Contact Information
- Complete contact profile
- Communication preferences
- Associated company information

#### Activity Timeline
- All interactions with this contact
- Chronological order (newest first)
- Activity types: calls, emails, meetings, notes

#### Associated Deals
- Deals linked to this contact
- Deal values and stages
- Quick deal creation

## 💼 Sales Pipeline

### Understanding the Pipeline

#### Default Stages
1. **Prospecting** (10% probability)
2. **Qualification** (25% probability)
3. **Proposal** (50% probability)
4. **Negotiation** (75% probability)
5. **Closed Won** (100% probability)
6. **Closed Lost** (0% probability)

### Creating Deals

1. **Deal Information**
   - Deal title (descriptive name)
   - Deal value (monetary amount)
   - Currency selection
   - Expected close date

2. **Pipeline Stage**
   - Select appropriate stage
   - Probability auto-fills based on stage
   - Manually adjust probability if needed

3. **Associations**
   - Link to contact (decision maker)
   - Link to company (organization)
   - Add description and notes

### Managing Deals

#### Visual Pipeline
- **Kanban Board** - Drag deals between stages
- **Deal Cards** - Quick overview of each deal
- **Stage Metrics** - Count and value per stage

#### Deal Details
- Complete deal information
- Associated contacts and companies
- Activity timeline
- Deal progression history

#### Deal Actions
- **Move Stage** - Progress through pipeline
- **Edit Details** - Update deal information
- **Add Activity** - Schedule follow-ups
- **Close Deal** - Mark as won or lost

### Pipeline Analytics

#### Key Metrics
- **Total Pipeline Value** - Sum of all open deals
- **Weighted Pipeline** - Probability-adjusted value
- **Average Deal Size** - Mean deal value
- **Win Rate** - Percentage of deals won

#### Stage Analysis
- Deal count per stage
- Average time in each stage
- Conversion rates between stages
- Bottleneck identification

## 📥 Data Import

### Supported Formats
- **CSV Files** - Comma-separated values
- **Excel Files** - .xlsx and .xls formats
- **Template Download** - Get the correct format

### Import Process

1. **Prepare Your Data**
   - Download the CSV template
   - Format your data according to the template
   - Ensure required fields are filled

2. **Upload File**
   - Navigate to Contacts
   - Click "Import" button
   - Drag and drop your file or browse

3. **Configure Import**
   - **Skip Duplicates** - Ignore existing contacts
   - **Update Existing** - Overwrite existing data
   - Review import options

4. **Review Results**
   - Import summary with counts
   - Error report for failed records
   - Success confirmation

### Import Tips

#### Data Preparation
- Use the provided template
- Clean your data before import
- Remove duplicate entries
- Validate email addresses

#### Field Mapping
- First Name and Last Name are required
- Email should be unique
- Phone numbers can include formatting
- Company names should be consistent

#### Error Handling
- Review error messages carefully
- Fix data issues and re-import
- Contact support for complex issues
